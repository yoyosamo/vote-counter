# Vote Counter

Created as a Year 10 IT project, this is an election vote counter.  
The visual interface allows for the counting and anlysis of an election or vote, making it easy to implement more complex voting systems and, thus, fairer votes.  
Guides the user through the process of determining a winner through various voting systems.

_As an early, time restricted, school project the code may be quite... shocking._  
_But I liked the idea, and might try to make it better - so here it is for posterity (or at least the time being)_
